# WizardGame
Funny wizard game I made in comp sci class. I forget what the assignment was, but I got carried away and added more than necessary.

The wizard fights an imaginary enemy.
The user writes all the actions that they want the wizard to do in the main file ahead of time and runs the program to see the results.
The goal is to maximize damage dealt to the enemy without dying.
Luck is also a factor, so running the same set of actions will often yield different results, making it more difficult to find a perfect set of actions.
